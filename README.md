Orbit X Pro — Responsive Product Landing Page
A modern, fully responsive product landing page for Orbit X Pro, a premium smartwatch brand. Built as part of a Web Development Internship Minor Project (Week 02).
Technologies Used

HTML5
CSS3
Tailwind CSS
Flexbox
CSS Grid

Sections

Navigation Bar with logo and links
Hero Section with product image and CTA button
Features Section (4 feature cards in a grid)
Benefits Section with product highlights
Testimonials Section (3 customer reviews)
Pricing Section (Standard and Pro plans)
Footer with social links and contact info

Responsive Design
The page is fully responsive across mobile, tablet, and desktop screens using Tailwind CSS breakpoints.
How to View
Simply open index.html in any browser. No installation required.
